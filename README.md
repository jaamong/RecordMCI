# RecordMCI

MCI 환자를 챙겨야 하는 보호자가 단순하게 기록하기 위해 만들기 시작!  

### Notice 

개인 사용 용도 목적이며, 로컬에서만 동작하고 PC 브라우저 전용이다.

프로그램을 빠르게 제작하는 것이 목표이므로 매우 단순한 도메인 모델을 갖는다. 

주로 다음 목적을 위해 시작했다. 
- 콩을 섭취했는지, 블루베리는 섭취했는지 등 식품 섭취를 체크박스 형태로 기록
- 색칠공부나 기타 등 활동을 기록
  - e.g., 오늘 산책을 했는지, 했다면 얼마나 걸었고 몇분 동안 수행했는지.
- 특이 사항은 텍스트 메모 (최대 1,000자)
